Code

All software developed as part of this project is available in the repository below. This includes:

  • The Automatic Neuron Tracking System for Unconstrained Nematodes (ANTSUN) pipeline to extract neural traces from raw GCaMP volumes.
  • The C. elegans Probabilistic Neural Encoding Model (CePNEM) used to fit the neural traces and extract information about behavioral encoding.
  • Software for controlling the stage to track the worm as it is being imaged.
  • Example Jupyter notebooks for trace extraction, model fitting, model result interpretation, and plotting.
  • Additional downloadable data that can be directly imported into these notebooks, including trained neural network weights, raw posterior distributions from the CePNEM model, and more.

Note that this repository is static, so it is advisable to click on the links to the individual packages to ensure you are using the most up-to-date versions.
